Coloque aqui os prints (screenshots) dos sites publicados.

Nome do arquivo deve bater com o campo `screenshot` em `src/data/projects.ts`, por exemplo:

- `lies-of-p-wiki.png` → usado por `screenshot: '/projects/lies-of-p-wiki.png'`

Recomendado: imagem no formato 16:9 (ex: 1280x720), .png ou .jpg.
